# Phylogenetics

Coursework repository for phylogenetics analyses on an influenza dataset.

## Main Files
- `notebook-log.md` — notes and commands used for each homework
- `scripts/` — shell and R scripts for the workflow
- `results/` — selected output files
- `envs/` — conda environment file
- `data/metadata/` — sample IDs and metadata
- `analysis/` — analysis notes and code
